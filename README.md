# okitegami

生活の中に置いておく、短い手紙と小さな接続口。

## shelves

- `connect-hub/` — 接続先をオンオフする小さな操作盤
- `place-deck/` — 行きたい店と場所を、ルート付きカードにする

## place deck

`place-deck/` は、近くの行きたい店や場所をカード化して、徒歩・チャリ・電車/バスの Google Maps ルートを開くための local 向けページです。

画像欄はあとで自分で撮った写真を入れる前提のスロットです。公開運用する場合は、Google Maps のスクショではなく自前写真かプレースホルダーを使うのが安全です。
